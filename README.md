## Description

Base de datos epidemiológica completa sobre la evolución de casos de viruela por país durante los últimos 30 años (1994-2024). Incluye datos históricos pre-erradicación (1960-1980) y seguimiento post-erradicación con intervalos de 5 años.

Fuentes: Organización Mundial de la Salud (OMS), datos de vigilancia epidemiológica y documentos históricos de erradicación.

Variables: país, fecha, número de casos confirmados, tipo de viruela, fuente de reporte.
